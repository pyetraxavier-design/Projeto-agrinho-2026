# Tecnologia no agro

Projeto para o Concurso Agrinho 


•O Propósito e Objetivo do Projeto
O objetivo central deste trabalho é difundir o conceito de "Agrotech", demonstrando como a fusão entre a tecnologia digital e o campo é vital para o futuro da agricultura e da sustentabilidade, especialmente no contexto do estado do Paraná. Buscamos conscientizar o público — desde produtores rurais até a comunidade escolar — sobre a importância da modernização dos processos agrícolas. O foco está em provar que a tecnologia não serve apenas para aumentar os lucros, mas sim para otimizar o uso de recursos naturais, reduzir desperdícios e garantir a segurança alimentar das próximas gerações, alinhando-se perfeitamente aos valores de cidadania e preservação promovidos pelo Programa Agrinho.


 •O Conteúdo Disponibilizado no Site
A arquitetura de informação do site foi planejada para cobrir o tema de forma abrangente e de fácil navegação, dividindo-se nas seguintes frentes conceituais:
Interface de Boas-Vindas: Um banner principal focado no impacto da revolução digital no agronegócio, feito para engajar o visitante logo no primeiro contato.


 •Vitrine de Inovações Tecnológicas: Explicações detalhadas sobre as quatro principais ferramentas do campo moderno: o monitoramento por drones, a coleta de dados via sensores de Internet das Coisas (IoT), a eficiência do maquinário autônomo guiado por GPS e o cruzamento de dados meteorológicos com Big Data.
 

 •Central de Benefícios: Uma área dedicada a sintetizar as vantagens práticas da tecnologia, destacando o aumento real de produtividade, a redução de custos operacionais e a rastreabilidade dos alimentos.


 •Espaço de Sustentabilidade: Seção de extrema relevância para o projeto, focada em como a agricultura de precisão atua diretamente na preservação ambiental, economizando água e evitando a contaminação do solo.
 
 
 •Canal de Interatividade: Um formulário de contato funcional para receber dúvidas, feedbacks ou sugestões sobre a aplicação dessas tecnologias na comunidade local.
 
 
 •A Estética Visual Adotada
O design do site segue a linha "Orgânico Futurista". Essa escolha estética busca equilibrar dois mundos que andam juntos no agronegócio moderno:
 

 •Elementos Orgânicos: Uso de tons verdes profundos e formas suaves com bordas arredondadas que remetem à terra, à natureza e à preservação.
Elementos Futuristas: Utilização de fundos limpos e claros, fontes modernas do tipo sans-serif e o efeito de vidro jateado (glassmorphism) no menu superior, transmitindo a sensação de um sistema limpo, tecnológico e avançado.
 

 •As Ferramentas e Tecnologias Utilizadas
Para a construção técnica e desenvolvimento da aplicação, utilizamos as ferramentas padrão da tecnologia web atual:
 

 •Ambiente de Desenvolvimento: Todo o projeto foi codificado utilizando o Visual Studio Code (VSCode), aproveitando suas extensões de produtividade, como o Live Server, para testes de responsividade em tempo real.
 

 •Linguagem de Estruturação (HTML): Utilizada para erguer o esqueleto da página através de tags semânticas modernas. Isso garante que o site seja acessível e bem indexado pelos motores de busca.
 
  
  •Linguagem de Estilização (CSS): Responsável por aplicar toda a identidade visual orgânico-futurista, controlando o layout em grade (Grid Layout), o comportamento responsivo para celulares e as animações de transição.
 

 • Linguagem de Programação (JavaScript): Aplicada para trazer dinamismo e inteligência ao site, controlando o comportamento do menu conforme a rolagem da página e validando os dados inseridos no formulário de contato sem a necessidade de recarregar a tela.
